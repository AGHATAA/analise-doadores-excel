# 📊 Análise de Dados no Excel

Projeto desenvolvido para praticar recursos do Microsoft Excel voltados para análise e tratamento de dados.

## Objetivo

Realizar a organização e análise de uma base de dados utilizando ferramentas nativas do Excel.

## Recursos utilizados

- Microsoft Excel
- Tabelas
- Filtros
- SUBTOTAL
- COUNTA
- Organização de dados
- Análise de informações

## Funcionalidades

- Contagem total de registros
- Contagem dinâmica após aplicação de filtros
- Organização da base de dados
- Manipulação de informações utilizando fórmulas

## Aprendizados

Durante este projeto pratiquei:

- Manipulação de dados
- Utilização de funções do Excel
- Aplicação de filtros
- Organização de planilhas
- Análise de informações
